Globolo App will help any person to learn the basic words of different languages through their native language, for example, an Indian can learn Mandarin through his/her native languages like Tamil, Hindi or Kannada. Globolo is a venture that targets students, expatriates, tourist and language lovers. This project is purely a voluntary project involving various people of common interest. This App starts with offering 500 words in following languages initially.


Languages: English, Cantonese,  Mandarin, Japanese, Nepalese, Korean.

Indian Languages: Tamil, Hindi, Kannada, Malayalam, Sanskrit

We will be releasing the words in the following languages one after the other.

Indonesian, Tagalog, Urdu, Spanish, Bengali, Telugu, Rajasthani
